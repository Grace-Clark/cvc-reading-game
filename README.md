# CVC Reading Game

A browser-based reading-skills game adapted from a PowerPoint assessment.

- **Part 1 (10 questions):** Read the CVC word, pick the matching picture.
- **Part 2 (5 questions):** See the picture, pick the matching word.
- Includes 2 practice items beforehand to confirm audio is working.

Audio uses the browser's built-in Web Speech API — no audio files required.

## Run locally

Just open `index.html` in a browser. No build, no server, no dependencies.
